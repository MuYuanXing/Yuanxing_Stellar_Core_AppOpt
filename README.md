# 星核线程：可视化WebUI与云端配置生态

这是一个集成了可视化WebUI和云端配置生态的线程调度模块。我们的目标是让你能像使用手机自带功能一样，轻松、直观地管理应用的线程策略。

首先，诚挚感谢 **@SutoLiu** 的开源项目 **AppOpt**，本模块将同样遵循 **GPL** 开源许可。

---

## 核心功能概览

### 1. 智能 oiface 管理
考虑到 oiface 在部分设备和应用上可能表现不佳，我们在主页提供了两种管理模式：
*   **全局禁用**：一键关闭全局 oiface，简单直接。
*   **智能禁用**：模块会根据你的规则列表，在特定应用启动时自动禁用 oiface，退出应用后则自动恢复。

同时，主页下方提供实时服务日志，方便你快速定位和排查问题。

### 2. 强大且灵活的规则列表
*   **配置导入/导出**：支持导入他人的 `applist.conf` 文件作为本地配置，也可以轻松将自己的配置分享给朋友。
*   **优先级微调**：为每条规则独立设置 CPU 和 I/O 优先级。
*   **云端更新检查**：一键检测你从云端下载的配置是否有新版本，保持最佳性能。
*   **冲突智能处理**：针对多套云端配置并存的复杂情况，我们做了大量优化，确保规则不冲突，为你提供稳定流畅的体验。

### 3. 模块的灵魂：云端配置中心
云端是我们最核心的板块，它汇集了官方与社区的力量。
*   **多元化的配置来源**：我们会不定期上传经过精心调试的官方配置，作为基础或范例。但更重要的是来自社区——也就是每一位用户的分享。
*   **便捷的查找方式**：你可以按“游戏”、“日用”等分类浏览，或直接搜索处理器型号（如“骁龙8Gen3”）来找到心仪的配置。每个配置都附有下载量和评分，好坏由社区共同决定。
*   **高度自由的下载流程**：
    *   **下载方式**：选择“下载全部”获取完整配置，或“选择应用”按需挑选（例如，只导入《原神》的规则）。
    *   **应用方式**：
        *   **替换**：清空本地规则，完全使用新配置，适合重新开始。
        *   **合并**：保留本地规则，智能覆盖冲突项，并添加新规则。
        *   **追加**：仅添加本地没有的规则，不修改任何现有配置。

### 4. 成为分享者
我们鼓励每一位用户分享自己的配置。模块需要注册（不获取任何隐私），是为了让你在更换设备或刷机后，能通过恢复码找回账户，管理你上传过的配置。在“作者相关”页面设置昵称和密码后，即可上传你的作品。

### 5. 补充性能配置
除了线程绑核，你还可以在这里进行更细致的性能调校：
*   设定全局的 CPU 调速器或 I/O 调度器。
*   为特定应用配置专属的性能模式，与线程规则相辅相成。
